jQuery 라이브러리

용어 설명 예제
jQuery JavaScript를 기반으로 하는 빠르고 간결한 JavaScript 라이브러리 <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
선택자 (Selector) HTML 요소를 선택하는 방법을 정의하는 표현식 $(선택자).메서드(매개변수,매개변수)
메서드 (Method) jQuery 객체에 적용되는 동작 또는 기능을 정의하는 함수 $(선택자).hide();
이벤트 (Event) 사용자의 동작(클릭, 마우스 오버 등)에 대한 반응을 정의하는 동작 $(선택자).click(function() {
    alert('버튼이 클릭되었습니다.');
});
애니메이션 (Animation) 요소의 움직임, 페이드 등을 부드럽게 제어하는 기능 $(선택자).fadeIn();
AJAX 비동기적으로 서버와 데이터를 교환하고 웹 페이지를 업데이트하는 기능 $.ajax({
    url: 'data.php',
    type: 'GET',
    success: function(response) {
        alert('데이터를 성공적으로 가져왔습니다.');
    }
});


jQuery 메서드

카테고리 메서드 설명
속성 조작 메서드 attr() HTML 요소의 속성 값을 가져오거나 설정합니다.
prop() HTML 요소의 속성 값을 가져오거나 설정합니다.
val() 폼 요소의 값을 가져오거나 설정합니다.
data() 요소에 연결된 데이터를 가져오거나 설정합니다.
스타일 조작 메서드 addClass() HTML 요소에 클래스를 추가합니다.
removeClass() HTML 요소에서 클래스를 제거합니다.
toggleClass() HTML 요소에 클래스를 추가하거나 제거합니다.
css() HTML 요소의 CSS 속성 값을 가져오거나 설정합니다.
글자 조작 메서드 text() HTML 요소의 텍스트 콘텐츠를 가져오거나 설정합니다.
html() HTML 요소의 내부 HTML 콘텐츠를 가져오거나 설정합니다.
append() HTML 요소의 내부에 콘텐츠를 추가합니다.
prepend() HTML 요소의 내부의 시작 부분에 콘텐츠를 추가합니다.
이벤트 연결 및 제거 메서드 on() 이벤트 핸들러를 연결합니다.
off() 이벤트 핸들러를 제거합니다.
trigger() 이벤트를 트리거하여 실행합니다.
시각 효과 메서드 show() HTML 요소를 보이도록 표시합니다.
hide() HTML 요소를 숨깁니다.
toggle() HTML 요소의 가시성 상태를 전환합니다.
애니메이션 메서드 fadeIn() HTML 요소를 서서히 나타나게 합니다.
fadeOut() HTML 요소를 서서히 사라지게 합니다.
animate() HTML 요소에 대한 사용자 정의 애니메이션을 제어합니다.
